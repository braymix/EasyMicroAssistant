🤖 Knowledge Assistant

Assistente AI locale — Avvio Rapido

⏱ 20–30 min al primo avvio
Requisiti: Windows 10/11  ·  Minimo 8 GB RAM  ·  15 GB spazio libero  ·  Connessione internet (solo primo avvio)

Avvio in 5 Step

STEP 1

🐳 Installa Docker Desktop

  1. Vai su docker.com/products/docker-desktop
  2. Scarica e installa per Windows
  3. Se compare "Use WSL 2" → spunta ✅
  4. Riavvia il PC
  5. Aspetta che la 🐳 nel system tray diventi verde
STEP 2

📁 Scarica il Progetto

  1. Estrai lo ZIP ricevuto
  2. Mettilo in una cartella facile da trovare
    es. C:\Progetti\knowledge-assistant
STEP 3

⚙️ Avvia il Setup Automatico

  1. Apri la cartella scripts\
  2. Doppio click su setup-windows.bat
  3. Quando chiede la modalità → digita 1 e premi Invio
  4. Aspetta 10–15 min (download immagini + modello AI)
✅ Vedrai: "SETUP COMPLETATO CON SUCCESSO!"
STEP 4

🌐 Apri l'Interfaccia

  1. Apri il browser
  2. Vai su http://localhost:3000
  3. Clicca "Sign Up"
  4. Crea il tuo account
⚠️ Il primo account diventa admin.
I dati restano solo sul tuo PC.
STEP 5

🧠 Scarica un Modello AI

Se lo script non l'ha già fatto:

  1. Clicca "Select a model" in alto a sinistra
  2. Scrivi llama3.2:3b
  3. Clicca "Pull"
  4. Aspetta il download (~2 GB)

Usare la Knowledge Base

📚

1 — Crea la KB

Workspace → Knowledge → clicca + → dai un nome (es. "Byblos") → Create

📄

2 — Carica Documenti

Clicca + dentro la KB → Upload Files → scegli PDF, TXT, MD, DOCX

💬

3 — Usa in Chat

Scrivi # nel campo messaggio → seleziona la KB → fai la domanda

Problemi Frequenti

Problema Soluzione
Pagina non si apre Avvia Docker Desktop, aspetta balena verde 🐳, poi start-full.bat
Nessun modello disponibile Esegui pull-model.bat → scrivi llama3.2:3b → Invio
Risposte molto lente Usa un modello più piccolo: llama3.2:3b invece di 8b
KB non trova informazioni Controlla spunta ✅ sul file in Workspace → Knowledge
